RESHAPE

让跑调的那一遍,变好听

两段式:先修音准,再以你自己的声音重生成。成品在调、干净,且仍是你的音色。

i.给它三样东西
务必逐句对齐:「你的原唱」要和「想唱成的样子」在 总时长、每句歌词的起止时间、节奏型 上完全一致——唱快唱慢、断句或换气不同,都会让修音错位。先用同一段、卡同样的节拍录。
「想唱成的样子」请先去掉和声(可在「拆声」里得到 clean / lead),别直接用整首原曲。「声音样本」约 10–30s 干净清唱即可(超过 30s 会自动截取)。
ii.三个旋钮
音准精修150
快一点更精细
扩散步数 50–200 · 想快可降到 50
音色细腻度50
快一点更细腻
重塑步数 20–100 · 25 也够用
贴近你的声音0.7

i诚实说:「贴近你的声音」是个细微的旋钮⌄
这一版用的模型里,这个旋钮(cfg)的官方定位就是「subtle / 细微」。真正强力的相似度旋钮要更新一代的模型才有,这里没上。想更像你本人:与其拉这个旋钮,不如换一段更干净、更长的「你的声音样本」,再把「音色细腻度」调高一些,效果明显得多。
整首会自动分块、完整重塑(不截断);但越长、越精细就越慢——整首高精度可能要二十分钟以上,先拿副歌试试更快。

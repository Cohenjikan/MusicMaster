MusicMaster 设计稿 — 自托管字体
================================

为「纯本地运行 / 离线可用 / 不向第三方泄露访客 IP(GDPR)」,界面字体全部自托管,
不再热链 Google Fonts。以下三款均为 SIL Open Font License 1.1 (OFL),已随附各自许可证。

文件                         字体              许可证            来源
---------------------------- ----------------- ---------------- ------------------------------------------
fraunces-normal.woff2        Fraunces (VF)     OFL-1.1          github.com/undercasetype/Fraunces
fraunces-italic.woff2        Fraunces Italic   OFL-1.1          （经 fontsource 打包,Latin 子集,可变字重轴）
inter-normal.woff2           Inter (VF)        OFL-1.1          github.com/rsms/inter
jetbrains-mono.woff2         JetBrains Mono VF OFL-1.1          github.com/JetBrains/JetBrainsMono

许可证全文:OFL-Fraunces.txt / OFL-Inter.txt / OFL-JetBrainsMono.txt

中文不打包字体(CJK 体积过大):
  - 衬线(标题)走系统宋体:Songti SC / SimSun / STSong
  - 正文走系统黑体:PingFang SC / 微软雅黑 / Hiragino Sans GB
  在 Windows / macOS 上均原生可用,零下载。

字体仅作 woff2 资源加载,符合 OFL「可随软件捆绑分发、需随附许可证、保留版权声明」之要求。
